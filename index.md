# Project Memory

## Core Identity

**MedFlow Nexus** is an AI-powered healthcare operations platform designed to optimize hospital efficiency at both macro and micro levels.

Supports:

1. Regional Intelligence Mode  
   Tamil Nadu / district-wide healthcare monitoring, forecasting, and resource planning.

2. My Hospital Mode  
   Individual hospital private workspace for AI automation using hospital-specific data.

---

## Design System

- Dark-only premium theme
- Graphite background with teal/cyan accents
- Primary color: oklch(0.75 0.15 190)
- Background: oklch(0.13 0.02 260)
- Glassmorphism panels
- Subtle glow states
- Inter font
- Enterprise command-center UI
- No light mode

---

## Technical Stack

- TanStack Start
- Tailwind CSS v4
- Vite
- File-based routing in `src/routes/`
- TypeScript
- Modular service architecture

---

## Data Philosophy

- No fake random demo data
- Real public datasets where possible
- Tamil Nadu healthcare context
- Transparent pipelines
- Clearly labeled synthetic simulation streams for demos
- Explainable outputs only
